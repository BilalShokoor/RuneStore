# RuneStore

OSRS (Old School RuneScape) account marketplace project.

## Project Structure

- `market_analysis.md` - Competitive market analysis with pricing data from 10+ competitors
- `account_database.csv` - Raw pricing database with 35+ account listings across sources
- `.gitignore` - Standard ignore patterns

## Context

- This is a storefront for selling OSRS accounts
- Key competitors: Empirers (sell.app), RSOrder, EzRSAccounts, Eldorado.gg, PlayerAuctions
- Account categories: Starters, Pures, Mains, End-Game, Ironman/HCIM
- Price range spans $0.38 (basic starters) to $4,600 (ultra-premium end-game)
- Best volume segment: 1 Def Pures at $50-$170
- Best margin segment: Infernal Cape accounts and HCIM
- Jagex Launcher OTP login is now the market standard for account security
